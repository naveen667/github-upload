About Project - 
This project implements addition of numbers present in the string. The Add() method takes a string containing numbers separated by delimiters as input and return the sum of those numbers.

eg:
I/P - 1,2,3		O/P - 6

Prerequisites:
JDK 1.8
Maven 3

How to run:

Using ecliple - 
Import the project as maven project in eclipse.
Open StringAdditionTest.java.
Right click --> Run as --> JUnit test

Using maven - 
Ensure maven is installed and environment variables are set for maven.
Open cmd.
Go to project home directory where the pom.xml is present.
Run command mvn test
